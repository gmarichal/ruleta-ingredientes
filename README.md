# 🎡 Ruleta de Ingredientes

Juego web para sortear ingredientes de cocina (desayunos y meriendas, dulces y salados).

## Cómo se juega
1. Cada participante escribe su nombre y aprieta **¡Girar!**
2. La ruleta le asigna un ingrediente al azar (se elimina de la ruleta).
3. Al **Finalizar juego** se muestra la tabla Persona/Ingrediente, con opción de copiarla.

## Uso
Abrir `index.html` en cualquier navegador. No requiere instalación ni servidor.

Hecho con HTML, CSS y JavaScript puro (canvas).
